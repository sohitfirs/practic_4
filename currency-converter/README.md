Для запуска нужен установленный Node.js

Установка зависимостей:
cd backend
npm install

cd backend
npm install

Запуск:
    В первом терминале:
        cd backend
        npm run dev

    Во втором терминале не закрывая первый:
        cd frontend
        npm run dev

Перейти по адресу: http://localhost:5173
